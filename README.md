# Aryan Kumar - Portfolio

An interactive React portfolio showcasing skills in AIML, Deep Learning, and Computer Vision.

## Features

- **Typing Animation**: "Hey! I'm Aryan Kumar" appears with a typewriter effect
- **Interactive Design**: Hover effects on expertise tags and contact icons
- **Responsive Layout**: Works on desktop and mobile devices
- **Dark Theme**: Professional dark background with blue accents
- **Contact Integration**: Direct links to WhatsApp, LinkedIn, Facebook, GitHub, Email, and Location

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view in browser

## Customization

- Replace the profile image placeholder in `src/App.js`
- Update contact links and personal information as needed
- Modify colors and styling in `src/App.css`

## Build for Production

```bash
npm run build
```

This creates an optimized build in the `build` folder ready for deployment.
